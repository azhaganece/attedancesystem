Excel File Format Guide

Your Excel file should have these columns (first row as headers):

Column Name Required Example Notes
Name Yes John Doe Full name of student
Register Number Yes 2024CSE001 Unique registration number
Year Yes 1, 2, 3, 4 or 1st Year Auto-detected format
Department Optional CSE, ECE, IT, AIDS Uses your department if empty
Email Optional john@email.com Auto-generated if empty
Phone Optional 9876543210 10-digit phone number
Auto Year Sorting

Students will be automatically sorted into: 1st Year, 2nd Year, 3rd Year, 4th Year

Password Generation

Passwords are auto-generated from the last 6 digits of the register number.
Example: Register Number 2024CSE001234 = Password 001234
